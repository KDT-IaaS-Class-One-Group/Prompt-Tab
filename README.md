# Prompt-Tab
## **기술 스택**

- **서버**
  - Node.js, Express.js
- **프론트엔드**
  - Vanilla JavaScript, CSS, HTML
- **데이터 저장**
  - JSON 파일

## **코어 파일 구성**

- **서버**
    - **app.js**: Express.js 애플리케이션의 시작점
    - **routes.js**: HTTP 라우팅을 처리하는 파일
- **프론트엔드**
  - **index.html**: 웹 페이지의 HTML 템플릿
  - **index.css**: 웹 페이지의 CSS 스타일
  - **index.js**: 웹 페이지의 JavaScript 코드
